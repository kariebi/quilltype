# Changelog

All notable changes to this project will be documented in this file.

The format is inspired by Keep a Changelog and the project follows SemVer.

## [Unreleased]

### Changed

- Repository, npm publishing, and release automation are being finalized for the first public release.

## [1.1.0] - 2026-07-22

### Added

- Output modes for `axios-client`, `swr`, `zod`, and `json-schema`
- Warning-level contract change detection and machine-readable reports
- Migration guides, cookbook docs, troubleshooting guidance, and compatibility notes
- NestJS, remote schema, and monorepo examples
- Issue templates and a pull request template

### Changed

- Watch mode now prints clearer trigger and plan summaries
- The public repository no longer tracks the private product checklist file

## [1.0.0] - 2026-07-22

### Added

- Stable CLI commands: `init`, `generate`, `check`, `watch`, `config validate`, and `doctor`
- Config-first and flags-first workflows
- Output modes for `types`, `fetch-client`, and `react-query`
- Breaking-change detection for OpenAPI contracts
- Local and remote watch support
- Config schema and validation
- Example stacks for FastAPI, Express, Go, and Rust Axum
- Product-facing documentation and contributor guidance
